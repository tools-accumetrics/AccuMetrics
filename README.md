# AccuMetrics - Analytics Pixel Multi-tenant

Sistema de analytics con soporte para múltiples clientes y proyectos.

## Stack
- Backend: Vercel Serverless Functions (Python)
- Base de datos: Supabase (PostgreSQL)
- Frontend: JavaScript vanilla

## Tracking Codes
- Website Principal: [TU-TRACKING-CODE-1]
- Blog: [TU-TRACKING-CODE-2]
- App Móvil: [TU-TRACKING-CODE-3]

## Setup
1. Configurar Supabase
2. Desplegar en Vercel
3. Configurar variables de entorno
4. Insertar píxel con tracking_code

## Estructura
- `api/track.py` - Endpoint de tracking
- `pixel-tracking.js` - Píxel JavaScript
- `vercel.json` - Configuración de Vercel
